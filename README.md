
# Minimal Productivity App

This is a code bundle for Minimal Productivity App. The original project is available at https://www.figma.com/design/E7jWirCKDBgT0MOgwt2Gc2/Minimal-Productivity-App.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Building for Android APK

This project can be built as an Android APK using Capacitor.

### Prerequisites for local build

1. Java JDK 11 or higher
2. Android SDK
3. Node.js 18+

### Local Build Steps

#### Using npm scripts (recommended)

```bash
# Build web app and Android APK in one command
npm run build:android

# The APK will be generated at `android/app/build/outputs/apk/debug/app-debug.apk`
```

#### Manual steps

1. Install dependencies: `npm ci`
2. Build web app: `npm run build`
3. Sync with Capacitor: `npx cap sync` (or `npm run sync:android`)
4. Build Android APK:
   ```bash
   cd android
   ./gradlew assembleDebug
   ```
   The APK will be generated at `android/app/build/outputs/apk/debug/app-debug.apk`

### Using GitHub Actions

This repository includes a GitHub Actions workflow that automatically builds APKs on push to main/master branch.

1. Push your code to GitHub
2. Go to the "Actions" tab in your repository
3. The workflow will run and generate two APKs:
   - Debug APK: `minimal-productivity-app-debug-apk`
   - Release APK (unsigned): `minimal-productivity-app-release-apk`

You can download the APKs from the workflow artifacts.

### Installing the APK

1. Enable "Install from unknown sources" on your Android device
2. Transfer the APK to your device
3. Open the APK file to install

## Project Structure

- `src/` - React application source code
- `android/` - Android project (generated by Capacitor)
- `dist/` - Built web assets (generated by `npm run build`)
- `.github/workflows/build-apk.yml` - GitHub Actions workflow for building APKs
